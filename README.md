## 目录结构

```
morning/YYYY-MM/YYYY-MM-DD.md   # 每日 9:27 竞价速报
evening/YYYY-MM/YYYY-MM-DD.md   # 每日 15:05 盘后复盘
```

## 内容说明

### 早盘速报（morning）
- 大盘情绪评级
- 板块风向 Top3
- 主板强势竞价股 Top10
- 连板/情绪龙头识别
- 重点关注方向

### 盘后复盘（evening）
- 盘面概况（指数、涨跌比、成交额）
- 核心题材回顾
- 早盘推荐票表现
- 明日计划（触发条件 + 止损）
