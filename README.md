# 📊 Apex Capital Daily Reports

A股每日自动报告，由 Bocchi (AI) 生成。

## 目录结构

```
YYYY-MM/
  YYYY-MM-DD/
    YYYY-MM-DD-morning.md    # 每日 9:25 竞价速报
    YYYY-MM-DD-evening.md    # 每日 15:05 盘后复盘
  WEEK_W17.md                # ISO 周总结（周五 15:30 生成）
  WEEK_W18.md
  ...
```

## 内容说明

### 早盘速报（morning）9:25
- 大盘情绪评级
- 板块风向 Top3 + 领涨股
- 主板强势竞价股 Top10（排除创业板/科创板/北交所/ST）
- 连板/情绪龙头识别
- 重点关注方向 2-3个
- 推荐股票代码列表（供晚报胜率统计）

### 盘后复盘（evening）15:05
- 盘面概况（指数、涨跌比、成交额、情绪评级）
- 核心题材 Top3 + 持续性判断
- 早盘推荐胜率回顾
- 明日计划（标的 + 触发条件 + 止损）
- 明日情绪预判

### 周总结（WEEK_Www）每周五 15:30
- 本周大盘走势
- 本周最强题材/板块
- 本周推荐综合胜率
- 下周市场展望
- 本周之星 Top5

> 数据来源：东方财富，仅供学习参考，不构成投资建议。
